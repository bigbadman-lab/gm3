# GM3

GM3 is a Solana-first discovery surface for real-time meme attention.

## Philosophy
- Read-only
- No social features
- No wallets in-app
- No auth in-app
- No charts
- No history browsing
- Distribution > interaction

GM3 surfaces what traders are paying attention to right now.

## Surfaces
- iOS (React Native)
- Web (Lovable)
- Public API (read-only)

## Non-goals
If it adds interaction, identity, or opinion, it does not belong in GM3.
